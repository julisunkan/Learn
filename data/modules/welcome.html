Welcome to the Tutorial Platform

This is a comprehensive tutorial platform built with Flask that allows you to:

  • Learn at your own pace - Complete modules when convenient for you
  • Track your progress - See how far you've come with visual progress indicators
  • Take notes - Keep track of important concepts as you learn
  • Test your knowledge - Complete quizzes to reinforce learning
  • Bookmark modules - Mark important lessons for later review

Features

The platform includes:

"A modern learning experience with dark mode support, keyboard navigation, and comprehensive progress tracking."

For Learners:

  • 🎥 HD Video content with HTML5 player
  • 📝 Rich text lessons and content
  • 📊 Progress tracking with localStorage
  • 🌙 Dark/Light mode toggle
  • ⌨️ Keyboard navigation (← → arrows to navigate, Space to complete)
  • 🔍 Search functionality
  • ⭐ Bookmarking system
  • 📋 Personal notes per module
  • ❓ Interactive quizzes
  • 🎓 Certificate generation upon completion

For Administrators:

  • 🔐 Hidden admin panel (accessible at /admin)
  • 📝 Module management with drag-and-drop reordering
  • 🎨 Site customization (colors, fonts, branding)
  • 📤 Course import/export as ZIP files
  • 💾 Auto-save functionality
  • 📊 Student feedback collection

Get started by exploring the modules and tracking your progress!
